Sobre a Aplicação

Essa aplicação foi construída para oferecer uma experiência de compra de produtos, com funcionalidades para usuários comuns e administradores.

O login será implementado para diferenciar os perfis de usuários e administradores.

Projeto tem como base praticar e fortalecer o aprendizado em tecnologias de desenvolvimento FullStack

Tecnologias Utilizadas:

  • Frontend: HTML, CSS, JavaScript, Angular.
  • Backend: Java Spring Boot (API RESTful).
  • Banco de Dados: PostgreSQL.

Objetivos:

  1. Compreender o uso do Angular e Angular Material
    • Data binding
    • Diretivas
    • Diretivas Estruturais
    • Diretivas de Atributo
    • Serviços
    • Injeção de Dependências
    • Formulários e validação
    • Rotas
  2. Lidar com requisições e autenticações utilizando Spring Web e Spring Security
  3. Praticar o uso do banco de dados relacional PostgreSQL
  4. Aprimorar conhecimento em interfaces atraentes e responsivas com uso do css flexbox
  5. Compreender conceitos de criptografia e certificados
  6. Monitoramento avançado com ferramentas de log(Log4j?)
  7. Sistema de pagamento(Stripe?)
  8. (E-mail service?)
  9. Realizar deploy em uma VPS
